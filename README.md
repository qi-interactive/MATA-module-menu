MATA Module Menu
==========================================

Manages and groups modules for MATA applications.

Installation
------------

- Add the module using composer: 

```json
"mata/mata-module-menu": "~1.0.0"
```

-  Run migrations
```
php yii migrate/up --migrationPath=@vendor/mata/mata-module-menu/migrations
```
